Noizy
=====
Ambient sounds generator

### Dependencies:
mpv
tmux

### Running
Run ./download.sh to get files from noisli
Run ./play-tmux.sh [sounds] to open panels with sounds playing

## Sounds
- rain
- thunderstorm
- wind
- forest
- bonfire
- coffeeshop
- seaside
- train
- leaves
- waterstream
- water
- summernight
